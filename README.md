## Structure

These writeups document my approach to solving CTF challenges and machines. Each writeup includes:

- **Enumeration** — recon and scanning phase
- **Exploitation** — vulnerability analysis and attack vectors
- **Privilege Escalation** — steps to gain root/admin
- **Lessons Learned** — key takeaways and tools used

> All content is for educational purposes only. Writeups for active HTB machines are published only when retired.
